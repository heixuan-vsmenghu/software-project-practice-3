# 数据集说明

本实验使用 TensorFlow 官方示例花卉图片数据集 `flower_photos`。

| 项目 | 内容 |
|---|---|
| 数据集名称 | `flower_photos` |
| 数据来源 | `https://storage.googleapis.com/download.tensorflow.org/example_images/flower_photos.tgz` |
| 图片数量 | 3670 |
| 类别数量 | 5 |
| 是否提交完整数据集 | 不提交，Notebook 和脚本会自动下载 |

## 类别

| 标签 | 含义 |
|---|---|
| `daisy` | 雏菊 |
| `dandelion` | 蒲公英 |
| `roses` | 玫瑰 |
| `sunflowers` | 向日葵 |
| `tulips` | 郁金香 |

本实验使用公开示例数据完成 TensorFlow / Keras 训练、TFLite 转换和 Android 端接入验证流程。
